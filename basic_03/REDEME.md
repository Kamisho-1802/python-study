# Flappy Bird AI — NEAT で自動学習するゲーム

世代を重ねるごとに賢くなる AI の成長をリアルタイムで観察できる機械学習体験アプリです。

---

## セットアップ

### 必要環境
- Python 3.11 または 3.12（3.13以降は pygame 非対応）

### インストール

\```bash
pip install pygame neat-python
\```

### ファイル構成

\```
flappy_bird_ai/
├── main.py                 # エントリポイント
├── game.py                 # ゲームロジック
├── bird.py                 # 鳥クラス
├── pipe.py                 # パイプクラス
└── config-feedforward.txt  # NEAT 設定ファイル
\```

---

## 使い方

### AI 学習モード（メイン）

\```bash
python main.py
\```

50羽の鳥が一斉にプレイを開始し、世代ごとに進化していきます。  
画面左上に `GEN`（世代数）・`SCORE`（通過パイプ数）・`ALIVE`（生存数）が表示されます。  
ESC キーで終了。学習終了後、`best_genome.pkl` に最優秀個体が保存されます。

### 手動プレイモード

\```bash
python main.py --manual
\```

スペースキーまたはマウスクリックでジャンプします。

### 学習済み AI のリプレイ

\```bash
python main.py --replay best_genome.pkl
\```

保存した最優秀個体が自動でプレイします。

## 仕組み — NEAT アルゴリズム

### NEAT とは

**NeuroEvolution of Augmenting Topologies** の略。  
ニューラルネットワークの**構造と重みを同時に進化**させる手法です。  
強化学習のような報酬設計が不要で、「生き残った個体が次世代に遺伝子を渡す」という自然選択のモデルが直感的にわかりやすいです。

### 進化の流れ

\```
第1世代（ランダムな脳・50羽）
  ↓ ほぼ全員すぐ死ぬ
生き残った個体・高スコア個体の遺伝子を選別
  ↓ 突然変異 + 交叉
第2世代（少しだけ賢い脳）
  ↓ ...繰り返す
第N世代（パイプを無限に通れる脳）
\```

### AI が使うセンサー情報（入力 5 つ）

| 入力 | 内容 |
|------|------|
| 鳥の Y 座標 | 現在の高さ |
| 鳥の縦速度 | 上昇中か落下中か |
| 次のパイプまでの水平距離 | どれくらい先にあるか |
| 次のパイプ（上）の Y 座標 | 隙間の上端 |
| 次のパイプ（下）の Y 座標 | 隙間の下端 |

出力は1つ（0.0 より大きければジャンプ）。

### スコアの付け方（適応度関数）

\```
毎フレーム生存          → +0.1
パイプの隙間に近いほど  → +最大0.05（フレームごと）
パイプを1本通過         → +5.0
死亡                   → -1.0
\```

生き延びるほど、パイプをうまく通るほど高スコアになります。

### 期待される学習曲線

| 世代 | 期待される挙動 |
|------|---------------|
| 第 1〜3 世代 | ほぼ全員がすぐ墜落 |
| 第 5〜10 世代 | 数羽が最初のパイプを通過し始める |
| 第 15〜25 世代 | 複数のパイプを安定して通過できる個体が登場 |
| 第 30〜50 世代 | ほぼ無限にプレイできる個体が誕生 |

## 学習速度・難易度の調整

### 学習を速くしたい

**`game.py`** の FPS を上げる（画面描画が速くなり、1秒あたりの進化が増える）

\```python
FPS = 120  # デフォルト: 60
\```

**`config-feedforward.txt`** の個体数を増やす（多様性が上がり収束が速まる）

\```ini
pop_size = 100  # デフォルト: 50
\```

### 難易度を変えたい

**`pipe.py`** を編集：

\```python
PIPE_GAP = 120   # 隙間を狭く（デフォルト: 150）
PIPE_SPEED = 7   # パイプを速く（デフォルト: 5）
\```

### 学習が停滞したとき

**`config-feedforward.txt`** の突然変異率を上げる：

\```ini
weight_mutate_rate = 0.9   # デフォルト: 0.8
weight_init_stdev  = 3.0   # デフォルト: 2.0
\```

### よくある問題と対処

| 症状 | 原因 | 対処 |
|------|------|------|
| 全員同じタイミングでジャンプ | 初期重みの多様性不足 | `weight_init_stdev` を大きくする |
| 一向に学習が進まない | 適応度関数が機能していない | パイプ通過ボーナスを増やす |
| 特定世代で停滞する | 種が絶滅しかけている | `max_stagnation` を増やす |
| 画面が固まる | イベントループの問題 | ESC で終了して再起動 |
